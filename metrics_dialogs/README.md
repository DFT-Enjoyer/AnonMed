# Оценка разнообразия и живости медицинских диалогов

Скрипт `metrics_dialogs.py` вычисляет метрики для корпуса диалогов, хранящихся в формате JSONL.  
Позволяет быстро оценить, насколько сгенерированные диалоги разнообразны и естественны.

## Используемые метрики

Все метрики рассчитываются по репликам (без тегов `[S:...]`) из всех диалогов вместе.
Для их вычисления **не требуются эталонные диалоги** — анализ проводится только на вашем корпусе.

### Distinct-1 и Distinct-2
- **Что измеряют**: долю уникальных слов (Distinct-1) и уникальных пар слов (Distinct-2) среди всех слов/пар слов в репликах.
- **Интерпретация**:
  - Чем выше, тем разнообразнее лексика.
  - Для хорошего разнообразия на коротких диалогах обычно Distinct‑1 > 0.3, Distinct‑2 > 0.6.
- **Почему не TTR**: Type‑Token Ratio (TTR) очень чувствителен к длине текста. На коротких диалогах он часто совпадает с Distinct‑1 и не даёт дополнительной информации.

### Доля реплик с разговорными маркерами (interjection_ratio)
- **Что измеряет**: процент реплик, содержащих хотя бы одно междометие, хезитацию или разговорный маркер (`ага`, `угу`, `ну`, `э-э`, `ммм`, `так`, `ой`, `вот` и др.).
- **Интерпретация**:
  - Прямой индикатор естественности речи.
  - Для живого устного диалога обычно не менее 30% реплик содержат такие маркеры. Значения ниже 20% говорят о формальном, сухом стиле.

### Средняя длина реплики (avg_replica_length)
- **Что измеряет**: среднее количество слов в одной реплике.
- **Интерпретация**:
  - Диалоги с короткими репликами (5–10 слов) воспринимаются как более динамичные и естественные.
  - Длинные реплики (>15 слов) часто превращают диалог в монолог, снижая живость.

### Self-BLEU
- **Что измеряет**: среднюю степень схожести одного диалога со всеми остальными диалогами в корпусе (по словам).
- **Интерпретация**:
  - Высокий Self-BLEU (>0.5) означает, что диалоги строятся по одному шаблону, фразы повторяются.
  - Низкий Self-BLEU (<0.3) — диалоги разнообразны.
  - **Важно**: на очень коротких диалогах (2–4 реплики) Self-BLEU может быть завышен из‑за высокой вероятности совпадения общих фраз («здравствуйте», «хорошо», «спасибо»). Интерпретируйте с осторожностью.

## Требования и установка

- Python 3.7+
- Библиотека NLTK

pip install nltk
python -c "import nltk; nltk.download('punkt')"
