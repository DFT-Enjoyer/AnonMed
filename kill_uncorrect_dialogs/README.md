# JSONL Filter: Duplicates, @ Symbol, Repeated Blocks

Скрипт удаляет строки из JSONL-файла, если их текстовое поле (`text`) содержит:

1. Символ `@` в любом месте.
2. Два или более одинаковых **блока** (строки), полученных при разделении текста по `\n`.
3. Повторяющиеся слова или пары слов внутри одного блока (части текста между `\n`).

## Правила фильтрации

### 1. Символ `@`
Если в поле `text` присутствует `@`, строка удаляется.

### 2. Дублирование целых блоков (строк)
Текст разбивается на блоки по символу переноса строки `\n`. Если среди полученных блоков есть два полностью идентичных (включая пустые), строка удаляется.

Пример: `"Привет\nПривет\nКак дела"` → блок `Привет` повторяется → **удалить**.

### 3. Дублирование слов внутри блока
Каждый блок дополнительно проверяется на:
- **Два одинаковых слова подряд** (например, `а а`)
- **Повторяющуюся пару слов** (например, `a b ... a b`)

Если такое найдено хотя бы в одном блоке, строка удаляется.

Корректные строки сохраняются без изменений.

## Установка

Требуется Python 3. Стандартная библиотека, без внешних зависимостей.
