# translit.py

Транслитерация email‑адресов и ников Telegram (латиница → кириллица) с пробелами вокруг цифр в режиме `words`.

## Новое в этой версии
- В режиме `words` **цифры автоматически отделяются пробелами** от буквенных частей.
  - `john00123` → `джон ноль ноль двенадцать три`
  - `a01b` → `а ноль один б`
- Символ `@` по‑прежнему окружён пробелами: `собака`.

## Зависимости
Установите `num2words`:
```bash
pip install num2words

python translit.py {mode} input.csv output.csv [--columns COL1 COL2 ...]
