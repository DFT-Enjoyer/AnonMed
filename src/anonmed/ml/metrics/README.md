### Базовые метрики
- `example_count`: Возвращает количество предсказаний и количество кейсов в датасете.

### Entity Hard (строгое совпадение сущностей)
Объект сравнения — кортеж `(line_idx, begin, end, label)`. Требуется точное совпадение границ и класса.
- `entity_hard_precision`: Доля сущностей с полностью верными границами и классом среди всех предсказанных.
- `entity_hard_recall`: Доля сущностей с полностью верными границами и классом среди всех эталонных.
- `entity_hard_f1`: F-мера (гармоническое среднее precision и recall) для строгого совпадения сущностей.
- `entity_hard_accuracy`: Отношение числа верно предсказанных сущностей к общему числу затронутых (TP / (TP + FP + FN)).

### Entity Soft (частичное совпадение сущностей)
Объект сравнения — сущности с одинаковым `label` и пересекающимися границами. Сопоставление «один к одному» выполняется через алгоритм максимального паросочетания в двудольном графе.
- `entity_soft_precision`: Доля предсказанных сущностей, успешно сопоставленных с эталонными.
- `entity_soft_recall`: Доля эталонных сущностей, успешно сопоставленных с предсказанными.
- `entity_soft_f1`: F-мера для частичного совпадения сущностей.
- `entity_soft_accuracy`: Отношение числа успешно сопоставленных сущностей к общему числу уникальных предсказанных и эталонных сущностей.

### Char Hard (посимвольное сравнение с учетом класса)
Объект сравнения — кортеж `(line_idx, char_idx, label)`.
- `char_hard_precision`: Доля символов с верно предсказанным классом среди всех предсказанных.
- `char_hard_recall`: Доля символов с верно предсказанным классом среди всех эталонных.
- `char_hard_f1`: F-мера для посимвольного совпадения с учетом класса.
- `char_hard_accuracy`: Отношение числа верно классифицированных символов к общему числу затронутых разметкой символов.

### Char Soft (посимвольное сравнение без учета класса)
Объект сравнения — кортеж `(line_idx, char_idx)`. Оценивается исключительно качество выделения границ.
- `char_soft_precision`: Доля предсказанных символов, попавших внутрь границ эталонных сущностей.
- `char_soft_recall`: Доля символов эталонных сущностей, захваченных предсказаниями.
- `char_soft_f1`: F-мера для посимвольного совпадения границ.
- `char_soft_accuracy`: Отношение числа совпавших по границам символов к общему числу затронутых символов.

### Coverage (покрытие)
- `coverage_percent`: Процент символов эталонных сущностей, найденных моделью (пересечение множеств предсказанных и эталонных символов).
- `over_coverage_percent`: Процент лишних предсказанных символов (False Positives) относительно общего объема эталонной разметки.
