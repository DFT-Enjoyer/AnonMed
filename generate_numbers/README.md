# generate_all_csv.py – генератор полносвязного датасета персональных данных

Скрипт создаёт CSV‑таблицу, в которой каждая строка содержит **11 типов персональных данных**, сгенерированных случайным образом.  
Данные представлены в **устной (verbal)** форме, а при использовании флага `--include-raw` дополнительно добавляются колонки с **цифровыми (raw) значениями**.

## Требования

- Python 3.10+
- Установленные зависимости из `requirements.txt` (пакет `num2words` и др.)
- Пакет `russian_data_generator` должен быть доступен для импорта (лежит в одной папке со скриптом или установлен)

## Использование

```bash
python generate_all_csv.py --count N --output ИМЯ_ФАЙЛА.csv [--include-raw]
