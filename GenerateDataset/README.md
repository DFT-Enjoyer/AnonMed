Основная логика для генерации - в файле scr_generate_llm_requests.py. Он использует датасет combined_by_columns.csv", это общий датасет для всех классов. Также есть сценарии - scenarious_from_max.txt. Передает полностью строчку датасета, а также решается, использовать ли сценарий или нет.
Итого генерируется какой - то набор диалог, что-то нужно обработать руками. Получаются на выходе два файла - набор txt диалогов, а другой уже размеченная версия ткстишных.
После этого из диалога вытаскиваются поддиалоги скриптом extract_subdialogs.py, в результате получается обычные txt и размеченные. Это и будет датасет.
